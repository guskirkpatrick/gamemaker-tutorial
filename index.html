What is Gamemaker Studio 2?

Gamemaker Studio 2 or Gamemaker, is a development environment for 2D games. It has a comprehensive list of tools for designing games and is very user friendly. It features both a drag and drop style of programming and its own programing language called game maker language, for this tutorial I will be using game maker language (GML)
